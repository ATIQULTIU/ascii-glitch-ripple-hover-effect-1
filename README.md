# ✨ ASCII Glitch Ripple Hover Effect

An experimental **ASCII ripple hover animation** built with **HTML, CSS, and JavaScript**.  
When a user moves the cursor over text links, a ripple wave travels across the characters, transforming them into dynamic ASCII symbols and creating a **glitch-inspired interactive effect**.

This project explores **creative UI animation** and **modern micro-interactions** that can enhance portfolios, landing pages, and experimental web designs.

---

## 🚀 Features

✨ Interactive **ASCII ripple hover animation**  
⚡ Smooth **wave-based glitch transition**  
🧩 Built with **pure Vanilla JavaScript**  
🎯 Lightweight and easy to integrate  
📱 Minimal, responsive layout  
🎨 Perfect for **creative portfolios and UI experiments**
